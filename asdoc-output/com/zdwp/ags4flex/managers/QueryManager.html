根据当前查询参数设置，执行相应 Query 查询。

Parameters

Complete:Function — 查询成功回调的函数。
 
Fault:Function (default = null) — 查询失败回调的函数。
 
type:String (default = NaN) — 查询过程类型，QUERY_FEATURE：基础要素查询；QUERY_COUNT：查询要素总数；QUERY_IDS：查询要素ObjectId集合。

Identify 查询参数设置。

Parameters

setQueryParams()method 
public function setQueryParams(map:Map, layer:FeatureLayer, queryCondition:Object, objectIds:Array = null, outFields:Array = null, orderByFields:Array = null, returnGeometry:Boolean = true):void

Query 查询参数设置。

Parameters

map:Map — 地图对象。
 
layer:FeatureLayer — 查询的目标子图层（FeatureLayer）。
 
queryCondition:Object — Query对应的查询条件，若为Geometry查询则该参数应该为一个Geometry对象，若为Text关键字查询，该参数应该为一个where条件。
 
objectIds:Array (default = null) — 查询限定的要素ObjectId集合。
 
outFields:Array (default = null) — 查询结果输出的字段数组。
 
orderByFields:Array (default = null) — 查询结果的排序字段数组。
 
returnGeometry:Boolean (default = true) — 查询结果是否返回Geometry信息，默认为true。

     //执行查询操作
     QueryManager.getInstance().executeQuery(onResult, 0nFault);

     }
