下面代码演示如何使用 LayerListControl 类：

首先，引用命名空间 xmlns:ags="http://www.zdwp.net/2013/flex/ags"，调用如下：

	<ags:LayerListControl
		Properties
		map="{map}"
		configXML="{configXML}"
		useLayerFadeEffect="false"
		Styles
		width="100%" 
		height="100%" />
	
	其中，map 为地图 Map 的引用，configXML 为系统配置文件 XML 对象.
