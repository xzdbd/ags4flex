分页组件（适用于空间分析查询QueryTask）.

MXML SyntaxShow MXML Syntax
Hide MXML Syntax

下面代码演示如何使用 GISPageControl 类：

首先，引用命名空间 xmlns:ags="http://www.zdwp.net/2013/flex/ags"，调用如下：

	<ags:GISPageControl
		Properties
		initFunction="{initQuery}"
		updateFunction="{updateQuery}"
		hostComponent="{waterLevelDatagrid}"
		pageCtrlToolsVisible="true"
		pageDetailInfoVisible="true"
		recordIds="{allObjectIds}"
		pageSize="10"
		
		Styles 
		width="100%" />
	
	其中，initQuery 和 updateQuery 分别是供分页控件内部调用的初始化和更新分页数据源的方法名，allObjectIds 是当前分页控件绑定数据表的所有记录集对应的 ID 集合.
