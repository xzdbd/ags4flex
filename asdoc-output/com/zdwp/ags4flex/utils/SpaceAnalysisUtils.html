executeLineCut()method 
public static function executeLineCut(lineCutParams:LineCutParameters, Complete:Function, Fault:Function = null):void

执行线段裁剪。 根据参数提供的线要素对象、起点和终点断点等，裁剪出起点和断点之间的线段。

Parameters

lineCutParams:LineCutParameters — 线段裁剪参数对象，包含Map对象、线要素对象、起点和终点断点等。
 
Complete:Function — 调用成功完成时的回调函数，该函数形式为： （resultLine为裁剪的线段对象，endPoint为裁剪结果的真实终点，info则包含一些操作信息） 
function Complete(resultLine:Polyline, endPoint:MapPoint, info:String);
 
Fault:Function (default = null) — 调用失败时的回调函数。

offsetPolylines()method 
public static function offsetPolylines(polylines:Array, distance:Number, Complete:Function):void

对线要素集进行偏移操作(OffSet)，为每一个线要素产生一条与之平行的线条。

Parameters

polylines:Array — 需要进行操作的线要素集合。
 
distance:Number — 偏移的位移量，如果为正值将在原始线要素右侧构造新的曲线，为负值则在左侧构造偏移曲线。
 
Complete:Function — 偏移操作完成时的回调函数。
